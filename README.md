# im-website-assets

Public staging assets for Inverloch Marine website spreadsheets, CMS import preparation, and preview links.

## Folders

- `images/` - image assets used for spreadsheet thumbnails and CMS import staging.
- `videos/` - video assets when public preview URLs are needed.
- `docs/` - PDF, brochure, and document assets when public preview URLs are needed.

## Public URLs

Files in this repository are published through GitHub Pages:

`https://teabone113.github.io/im-website-assets/<path-to-file>`

Example:

`https://teabone113.github.io/im-website-assets/images/boat-models/560-game-king/test-560-game-king-hero.jpg`
